# example_package

This example package shows how to create a cmake based c++ project and a simple
ros package.

* [simple_cpp_project](https://github.com/bit-ivrc/example_package/tree/master/simple_cpp_project) A simple c++ project template in linux. 
* [simple_ros_node](https://github.com/bit-ivrc/example_package/tree/master/simple_ros_node)  A simple ros package template in linux.
